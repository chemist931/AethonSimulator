# AethonSimulator
Aethon Spacecraft Simulator

Simulates the _Aethon_ spacecraft, a ~2060s-80s spacecraft designed to transport supplies to a lunar colony and Helium-3 back, for use in fusion reactors on Earth

*Requirements*
-Java 8
